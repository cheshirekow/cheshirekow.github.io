1 C++ Bindings for freetype {#cpp_freetype_page}
2 ==========
3 
4 cpp_freetype is a C++ wrapper around the FreeType2 library. It is a simple,
5 thin wrapper that doesn't do much on it's own. I wouldn't consider it
6 very useful except to vain C++ programs who want the font config objects
7 to be nicely namespaced and to eliminate the FT_ macros from their code.
8 
9 cpp_freetype mostly just provides objects which wrap freetype pointers
10 with the methods of the underlying object.
11 
12 Note-to-self: You probably want to link against this library using
13 Link Time Optimization (LTO) (i.e. the -flto switch with gcc). It can
14 inline functions that are declared in this library in the code that
15 uses it.
16 
17 As an example, here is a simple program which loads a font file and
18 displays some information from it, and then laods the outline for the
19 character 'A' and prints the list of path segments:
20 
21 @include test/cpp_freetype/tutorial/main.cpp
