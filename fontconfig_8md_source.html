1 C++ Bindings for fontconfig {#cpp_fontconfig_page}
2 ==========
3 
4 cpp_fontconfig is a very thin c++ wrapper around libfontconfig. It wraps
5 only the public API and takes advantage of the fact that most of the
6 library exposes opaque pointers.
7 
8 Most of the classes in cpp_fontconfig are wrappers around these opaque
9 pointers. As such, you should treat objects of these classes as if they
10 were, in fact, pointers themselves. It is probably better to pass around
11 copies and/or references to these objects rather than pointers to
12 these objects.
13 
14 As a simple demonstration, here is a program which will output a system
15 font file which matches a requested family name.
