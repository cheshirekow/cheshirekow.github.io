1 openbookfs {#openbookfs_page}
2 ==============================
3 
4 The openbook filesystem is a fuse-based file synchronizer. Once the fuse
5 filesystem is mounted file synchronization happens in the background. We're
6 planning to use a Bayou-like peer-to-peer synchronization strategy, however
7 at this point in time we're developing the system to use a dedicated server.
