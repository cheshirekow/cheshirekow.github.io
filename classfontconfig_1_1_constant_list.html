Detailed Description

utility class for building a static list of ObjectTypes, replaces static allocation of a list of FcObjectType objects

Example usage:

ConstantList list = ConstantList::create()
( "NameA", "ObjectA", 1001 )
( "NameB", "ObjectB", 1002 )
( "NameC", "ObjectC", 1003 )
