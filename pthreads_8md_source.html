1 cpp-pthreads {#cpp_pthreads_page}
2 ================
3 cpp-pthreads is a simple wrapper for pthreads which provides a slightly
4 more "modern", object oriented C++ interface to pthreads. It doesn't
5 provide any additional functionality, it simply makes multi-threaded
6 code slightly more readable in C++.
7 
8 Attribute objects can be filled using the stream operator. For instance:
9 @code
10  #include <cpp-pthreads.h>
11  #include <iostream>
12 
13  void* routine(){
14  std::cout << "Hello\n";
15  }
16 
17  int main(){
18  using namespace pthreads;
19 
20  const unsigned int stackSize = 100;
21  char stack[stackSize];
22 
23 
24  Attr<Thread> attr;
25  attr << JOINABLE
26  << Stack(stack,stackSize)
27  << DETACHED
28  << EXPLICIT
29  << GuardSize(10)
30  << SYSTEM;
31 
32  Thread thread;
33  thread.launch( attr, routine, 0 );
34  thread.join
35  }
36 @endcode
37 
38 
39 Mutexes are slightly easier to maintain with by using ScopedLocks
40 
41 @code
42  using namespace pthreads;
43  Mutex mutex;
44 
45  mutex.init();
46  /* lock scope */ {
47  ScopedLock lock(mutex); //< locks the mutex
48  //... (Do something)
49 
50  } // lock goes out of scope and releases the mutex
51  @endcode
52 
53 The cmake script for this library produces a shared library
54 (libcpp-pthreads.so) and a static library (libcpp-pthreads_static.a). The
55 install target for the makescript will install the headers, the libraries,
56 and a cmake find script FindCPPThreads.cmake.
